So, what is HTML?

HTML stands for Hyper Text Markup Language. HTML is the backbone of web pages, the structure that makes up what we see on the web.

HTML is a language that is written into a file that is able to be displayed on the web. It is made up of elements, or tags, that tell the web browser how to display the information.

Some examples of tags include p, which starts a new paragraph, h1, which tells the browser to show the text as a Header, and em, which places emphasis on the text.

HTML was developed in 1990 by Tim Berners-Lee.

We know that HTML is the basic make up of the web pages but also works with other languages like CSS to make what is seen today in most webpages.
